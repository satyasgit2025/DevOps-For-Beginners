Go to the location of .sh file.
```
cd /tmp
```
To see the list files and directories in the current directory.
```
ls -ltr
```
Adds execute permission to the file for everyone without changing existing read/write permissions.

```
chmod a+x xfusioncorp.sh
```
overwrites permissions with exact read/write/execute settings (Optional).
```
chmod 755 xfusioncorp.sh
```

Method-2: Without going in directory or location of file 
```
ls -l /tmp/xfusioncorp.sh
```
```
chmod a+x /tmp/xfusioncorp.sh)
```
